# Três vozes - O Jornal do Conhecimento Compartilhado

## Para rodar o projeto:

1. Clone o repositório

2. Instale as dependências python com o pip

```bash
pip install -r requirements.txt
```

3. Rode o projeto

```bash
python main.py
```
